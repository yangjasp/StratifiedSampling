# MultipleParameters
This repository contains the code accompanying the paper "Improving optimal subsampling through stratifiction". 
The file "ossat_functionsMarksAnglin2025.R" was reproduced directly from the supplemental materials of Marks-Anglin et al. (2025). This code is not original to the present manuscript and is included solely to ensure reproducibility of the analyses. All credit for this code belongs to the original authors.
